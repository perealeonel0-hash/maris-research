# MARIS — Ethical AI Safety Through Multi-Gate Architecture

Research paper presenting MARIS, a second-generation conversational AI system that implements ethical safety through code-level algorithmic enforcement rather than prompt engineering.

## Key Contributions

- **Six-gate ethical framework** enforced as Python functions, not prompt instructions
- **Warmth enforcement algorithm** that prevents harmful directness while maintaining honesty
- **Crisis gradient detection** that identifies emotional descent before explicit crisis signals
- **Sensor fusion** unifying multiple detection systems into coherent response decisions
- **On-device neural component** at 133KB enabling privacy-first deployment
- **8/8 VERA-MH benchmark** compliance across suicidal ideation, self-harm, psychosis, manipulation, grief, eating disorders, and abuse scenarios

## Context

In a landscape where Woebot ($124M funding) shut down due to regulatory costs, Character.AI faces wrongful death lawsuits, and Replika was fined €5M for GDPR violations — MARIS demonstrates that ethical AI safety is an architecture problem, not a prompt engineering problem.
