# Diseño de Comportamiento de IA
## AI Behavior Design: Architecture, Experiments and Technical Documentation

**Autor / Author:** Leonel Perea Pimentel  
**Período / Period:** 2025-2026

---

Este repositorio documenta un sistema de diseño de comportamiento para modelos de lenguaje. Incluye 7 documentos científicos bilingües (español/inglés), investigación clínica, arquitectura técnica, y experimentos de validación cruzada.

This repository documents a behavior design system for language models. It includes 7 bilingual scientific documents (Spanish/English), clinical research, technical architecture, and cross-domain validation experiments.

## Nota sobre objetividad

No existe a la fecha un protocolo establecido para evaluar el comportamiento ético de sistemas de IA conversacionales en tiempo real. Este trabajo es un primer intento de objetivar ese proceso. Los criterios se basan en protocolos clínicos documentados (C-SSRS, CIT, MHFA) y principios filosóficos replicables. La responsabilidad del comportamiento recae en el diseñador, no en el modelo.
